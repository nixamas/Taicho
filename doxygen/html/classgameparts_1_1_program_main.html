Detailed Description

This panel lets two users play checkers against each other. Red always starts the game. If a player can jump an opponent's piece, then the player must jump. When a player can make no more moves, the game ends.

The class has a main() routine that lets it be run as a stand-alone application. The application just opens a window that uses an object of type Checkers as its content pane.

There is also a nested class, Checker.Applet, that can be used as an applet version of the program. The applet size should be 350-by-250 (or very close to that).
